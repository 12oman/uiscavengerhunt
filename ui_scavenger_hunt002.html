UI Scavenger Hunt

Modal Window

A pop-up window overlaying the main content.

Floating Action Button

A circular button floating above the UI, often found in mobile apps.

Tooltip

A small pop-up box that provides information about a button or link when hovered over.

Card UI Design

A rectangular segment that presents information in bite-sized chunks.

Skeleton Loading

A user-friendly way to handle loading states by displaying a placeholder content structure.

Masonry Grid Layout

A grid layout based on columns, making it look like a masonry brick wall.

Badge Indicator

A small notification bubble often found on app icons or menu items.

Toggle Switch

A UI element that allows users to switch between two states, like on/off.

Off-canvas Menu

A navigational menu that slides in from the side, commonly used in mobile design.

Pagination

Navigation links to move through multi-page content.

Hero Image

A large, prominent image prominently placed on a web page, typically front and center.

Sticky Header

A header that remains at the top of the page even when users scroll.

Image Carousel/Slider

A rotating set of images, often used on homepages.

Accordion

A collapsible list of items that expands when one of the items is clicked.

Breadcrumbs

A secondary navigation scheme that reveals the user's location on a website.

Tabbed Content Boxes

Sections of content displayed by toggling through tabs.

Responsive Tables

Tables that adapt to different screen sizes, often by collapsing columns or turning rows into cards.

Mega Menu

Large dropdown menus typically used on e-commerce or news websites.

Lazy Loading Images

Technique where images load as they enter the viewport.

Fullscreen Video Background

A video playing in the full background of a webpage segment.
